# Proyecto Final - API Node.js & Firestore

Esta es una API REST desarrollada con Node.js y Express para la gestión de productos, integrada con Firebase Firestore como base de datos en la nube.

## 🚀 Tecnologías utilizadas
* **Backend:** Node.js, Express (ES Modules)
* **Base de Datos:** Firebase Firestore
* **Herramientas de prueba:** Thunder Client / Postman
